# Variabel dan Tipe Data

## Apa itu Variabel?
Variabel adalah tempat menyimpan data di Python. Lo gak perlu deklarasi tipe data—Python otomatis deteksi.

## Contoh Kode
```python
nama = "Andi"        # string (teks)
umur = 17            # integer (angka bulat)
ipk = 3.75           # float (angka desimal)
aktif = True         # boolean (True/False)

print(f"Nama: {nama}, Umur: {umur}")
